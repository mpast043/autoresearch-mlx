# A Tedious Manual Data Reconciliation Process

## Problem Statement
We pay $340/month for a tool that saves us maybe 2 hours. Keeping it anyway.

## Value Hypothesis
Reduce the recurring workflow failure before replacing the entire system.

## Audience
A tedious manual data reconciliation process that requires focused attention on boring repetitive work.

## Installation and Setup
1. Clone this repository:
   ```sh
   git clone https://github.com/your-repo-url.git
   cd a-tedious-manual-data-reconciliation-process-that
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. Run the script:
   ```sh
   make run
   ```
4. For running, ensure `db_path` is set correctly in your environment.
